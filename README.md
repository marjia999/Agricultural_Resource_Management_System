# Agricultural Resource Management System

AgriRMS is a full-stack web application designed to manage agricultural resources. It provides separate interfaces for administrators and clients to manage resources, submit service requests, schedule logistics, and process payments.

## User Roles

Admin - Has access to centralized dashboard with system analytics, can manage agricultural resources (add, edit, delete), process and approve service requests, assign resources and schedule logistics, generate invoices and manage billing, and view all registered clients.

Client - Can view personal dashboard, submit service requests for resources, track request status from Pending to Approved to Assigned to Completed, make payments via Card or PayPal, and update profile information.

4. Admin Dashboard: Real-time statistics showing total resources, clients, and service requests with quick actions for common tasks and recent activity monitoring.

2. Resource Management: Add new resources with name, type, price, and quantity. Update resource status between Available, In Use, and Maintenance. Track maintenance schedules and delete resources when needed.

3. Service Request System: Clients can request resources with specified quantity and duration. Admin approves and processes requests with automated price calculation based on daily rate multiplied by quantity multiplied by days.

4. Logistics & Scheduling: Assign available resources to approved requests, schedule delivery dates, and track assignment status throughout the process.

5. Billing & Payment System - Automatic invoice generation with card payment integration and PayPal payment option. Complete payment status tracking for all transactions.
